# Web chat application

This is a web chat application which is divided in 2 parts:

- Server (this one): https://github.com/ivarsarner/chat-server
- Client: https://github.com/ivarsarner/chat-client

## Tech stack

- React
- Redux
- Websockets (Socket.io)
- Winston (logger)
- Typescript

## Run locally

- npm run dev
